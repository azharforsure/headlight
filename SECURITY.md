# Security policy

## Supported versions

Only the latest release line receives security patches.

## Reporting

Email security@headlight.dev with details. Do not open public issues for security vulnerabilities.

We aim to:

- Acknowledge within 48 hours.
- Provide remediation plan within 7 days.
- Publish a fix or mitigation within 30 days.

Coordinated disclosure credit is offered unless you request anonymity.
